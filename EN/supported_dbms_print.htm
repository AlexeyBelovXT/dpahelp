Supported DBMS

 
DPA supports x64 versions of the Microsoft SQL Server since 2008R2 and newer. Redactions Standard, Enterprise.
 
The system can be connected to MS SQL Express.
It is not recommended because of restrictions:
· the size of the database is up to 10GB.
· Can use to 1GB of memory and the 1st processor (kernel) even if is set much more.
 
DPA is developed with use of the ORM DataObjects.NET standard therefore works with:
·   Oracle 10g, 11g, 12c, 18c;
·   Microsoft SQL Server CE 3.5, 4.0;
·   Windows Azure SQL Database 2008 R2, 2012, 2014, 2016;
·   Postgre SQL 8.3 - 11.2;
·   Firebird 2.5, 3.0;
·   MySQL 5.1, 5.5, 5.7, 6.0, 8.0;
·   SQLite 3 - 3.27.2.
