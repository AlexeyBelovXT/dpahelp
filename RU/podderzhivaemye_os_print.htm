Поддерживаемые ОС

 
Компоненты DPA устанавливаются на 64хразрядные версии следующих операционных систем MS Windows:
·     Windows Server 2016 Standard
·     Windows Server 2016 Datacenter
·     Windows Server 2012 R2 Standard
·     Windows Server 2012 R2 Datacenter
·     Windows Server 2008 R2 Standard
·     Windows Server 2008 R2 Datacenter
·     Windows 10
·     Windows 7
 
Так же требуется  NET Framework 4.6.2 (https://www.microsoft.com/ru-ru/download/details.aspx?id=53344).
