Классификация простоев

 
Для корректной работы в модулях "Оператор" (управление причинами простоя – для ввода причин простоя оператором) и "Аналитика" у пользователя должны быть настроены соответствующие права доступа к функциям и заполнены соответствующие справочники.
На панели навигации нажмите на кнопку , и в выпадающем списке выберите пункт «Оператор».
Главное меню. Стартовая страница
 
Главное окно модуля представляет собой операционную диаграмму состояний станков. Пользователю доступны только станки, на которые он назначен в качестве оператора. При возникновении события «Простой», система сигнализирует об этом пользователю сообщением «Обнаружен простой». Для регистрации причины простой выберите станок.
 
                    Оператор. Инициация регистрации простоя
Наведите курсор мыши на кнопку и нажмите на нее. В контекстном выпадающем меню выберите пункт "Простои".
 Оператор. Инициация регистрации простоя
 
С помощью фильтров на панели навигации можно регулировать отображение простоев. Для того чтобы указать причину простоя выберите требуемый простой из перечня.
 Пример отображения простоев за предыдущую смену
Укажите категорию простоя. Чтобы зафиксировать выбранную категорию, нажмите на кнопку .
