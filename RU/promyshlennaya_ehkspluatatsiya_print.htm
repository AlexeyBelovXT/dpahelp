Промышленная эксплуатация

Помимо рекомендованных требований советуем учитывать следующее:
·     Компоненты DPA (Host, Server, БД DPA) следует размещать по отдельности, на разные машины. В таком случае СУБД MS SQL устанавливается только на сервер с базой данных DPA.
·     Для сервера баз данных выделить дополнительно два жестких диска SSD по 300ГБ, объединенные в RAID1. Под «кэш» (временные базы для промежуточного хранения данных перед записью в основную).
·     Для DPA Host сетевое подключение со скоростью 10Гб/с
·     Подключать к одному DPA Server не более 50ти устройств (станков с ЧПУ). Если станков больше пятидесяти, то устанавливать дополнительный DPA Server.
·     Обязательно проконсультируйтесь с системным администратором по сетевым настройкам компонент DPA и устройств с ЧПУ. Грамотное планирование сетевых подключений влияет на общую производительность Системы.
 
 
Установка DPA под промышленную эксплуатацию требует тщательного планирования и настройки всех компонент. Рекомендуем обратиться к специалистам X-tensive за консультацией.
