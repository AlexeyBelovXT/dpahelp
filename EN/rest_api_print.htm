What allows REST API DPA
All functions implemented in DPA and available via the web interface to users, are also available through REST API. In other words, REST API allows to use all functionality of DPA.
Whether there is manual on REST API
REST API has no separate documentation yet. We specially gave methods intuitively clear names, organized information in the form of the structured list.
Customized a system which gives information on arguments of methods, and will also help to execute testing, to call methods, to receive the answer in a JSON format.
 
Security
REST API is available only to authorized users.
At the appeal to the REST API functions there are same checks of the rights and privileges which are carried out during the normal work in a system under the account of the corresponding user.
Authorization happens by method call/api/Account/Login.
