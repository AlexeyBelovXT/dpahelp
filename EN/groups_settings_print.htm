Groups settings

Click , and select "Security"
Select "Groups" tab.
                                                                                                                                                                        Security. Groups
 
By default in a system four user groups are created:
  •  Administrators
  •  Users
  •  Operators
  •  Control program managers
  •  Technologists
  •  Masters
To add new group click in a left corner of form.
                                                                                                                             Security. Add group
To manage membership, select group and click .
Select Group members tab. Click "Add" to add new members.
A member of a group can be a user, a group.
Select the users or groups you want to add and click OK
                                                                                                                                             Security. Membership of group
 
On the Group Membership tab you can set up which groups this group is a member of.
