Indicators

Parameters of monitoring can be stored in a DB SQL and in the database of real time.
By default, for prevention of growth of the database and decrease in load of the server, often in-out parameters, such as speeds of loading are stored in the database of real time.
Click , select "Enterprise structure".
 
To change this setting select an item of equipment, open indicators.

To select a necessary format of storage "In memory", "In a DB", "In the file system" - set a flag in the necessary menu item "History", and fill fields if necessary.
If indicator files folder is too big. Decrease parameter "Store (days)".
Also it's not recommended to store indicators data in database. System performanse is downgraded
