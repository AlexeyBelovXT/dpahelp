Recommended installation

· The DPA components (Host, Server, DPA DB) should be placed separately, on different machines. In that case MS SQL DBMS is installed only on the server with the DPA database.
· Addition two hard drives SSD on 300GB are needed for the database' cache. United in RAID1. "cache" - temporary bases for intermediate data storage before record in the main.
· For DPA Host network connection with a speed 10gb/s
· Connect no more than 50 devices (CNC machines) to one DPA Server. If more - set additional DPA Server.
· Surely consult with the system administrator on network settings the DPA component and devices with the CNC. Competent planning of network connections influences the overall performance of the System.
 
The DPA installation under commercial operation demands careful planning and setup of all a component. We recommend to address specialists X-tensive for consultation.
