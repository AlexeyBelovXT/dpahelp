Users actions

Click , and select "Security".
To navigation bars pass into the section "Actions of Users". In this section all actions of users of a system are displayed.
 
To track which actions were performed in a particular WS, you must copy part of its name to the address bar of the browser (after the domain name).
 
Paste the copied data into the search string.
 
 
After the name of the APM, we see which actions were performed, globally we can highlight the following actions and use them for filtering:
• Get - get information (passive operation, "view" operation);
• Set- indication of any previously not specified data;
• Add and Create - add and create an object
• Update - change any previously specified data;
• Remove - delete data;
• Login and LogOut - authorization actions.
