Предустановленным сервисным пользователем является auduser с паролем SUNRISE
 
Во время запуска станка можно увидеть такое диалоговое окно:
Чтобы зайти сервсиным пользователем, надо выбрать "Service". Еще один способ вызова окна входа: нажать кнопку <3> во время загрузки системы.
 
Указываете имя и пароль, после чего необходимо нажать кнопку соответствующего режима входа.
Сейчас две кнопки: "Сервисный рабочий стол" и "Сервисный центр". Если HMI установлен, то появится дополнительная кнопка "HMI".
 
если вы зашли в HMI пользователем auduser, вы можете внести изменения в сетевые настройки с помощью меню
Menu Select > Diagnosis > Bus TCP/IP > TCP/IP diagnostic
нажмите Change для внесения изменений.
Важно: менять надо настройки на X130 интерфейсе! Настройки не применяются сразу же. Потребуется перезапуск контроллера.
 
Примечание: Если в HMI зарегистрирован пользователь с правами оператора, то для смены пользователя надо выбрать меню
Menu Select > Diagnosis > Password > Set password
и указать пароль пользователя auduser
