Настройки системы

На панели навигации нажмите на кнопку , и в выпадающем списке выберите пункт меню «Настройка системы».
Главное меню. Стартовая страница
 
Данный раздел предназначен для установки пользовательских настроек системы.
В таблице ниже приведен полный перечень настроек и пояснений:
№ п/п
Раздел/Подраздел
                                                                   Назначение
1.
Интеграция
Настройки интеграции системы
1.1.
Импорт
Позволяет указать путь для импорта, сформировать ответы на внешние запросы и сигнализировать о состоянии
1.2.
Экспорт
Позволяет указать путь для выгрузки экспорта, сформировать URL очереди MSMQ для выгрузки, выгружать данные Renishaw и указать путь в файловой системе для выгрузки экспорта Renishaw
1.3.
Выгрузка заказов
Позволяет указать путь в файловой системе, сформировать URL очереди MSMQ для выгрузки, настроить выгрузку данных о старте, окончании и выполнении заказа
1.4.
Выгрузка ТОиР
Позволяет настроить выгрузку данных о ТОиР, указать путь в файловой системе и сформировать URL очереди MSMQ для выгрузки
1.5.
Выгрузка простоев
Позволяет настроить выгрузку данных о простоях, указать путь в файловой системе и сформировать URL очереди MSMQ для выгрузки
2.
Учёт простоев
Позволяет настроить длительность простоев и классификацию состояний простоев
3.
Отклонения от нормы выполнения
Позволяет настроить длительность отклонений и классификацию отклонений от нормы
4.
Хранение данных
Позволяет указать путь хранения лог индикаторов, данных вибромониторинга и медиа файлов
5.
Электронная почта
Позволяет указать пользовательские данные (для сохранения в системе), настроить входящие/исходящие параметры для почты
6.
СМС
Настройки для сервиса короткий сообщений
6.1.
Провайдер
Позволяет указать настройки параметров протокола провайдера для доставки сообщений
6.2.
GSM Modem провайдер
Позволяет указать настройки параметров СОМ-порта, Baud rate и таймаута
6.3.
HTTP провайдер
Позволяет указать URL отправки сообщений, имя параметра: номера телефона, текста сообщений и подписи отправителя
6.4.
SMPP провайдер
Позволяет указать параметры для протокола SMPP
6.5.
SMTP провайдер
Позволяет указать параметры для протокола SMТP
7.
Диспетчеризация
Позволяет управлять списком пользователей/группой пользователей и устанавливать настройки доступа
