Поддерживаемые СУБД

 
DPA поддерживает 64хразрядные версии Microsoft SQL server начиная с 2008R2 и выше. Редакции Standard, Enterprise.
 
Систему можно подключить к СУБД MS SQL Express.
Не рекомендуется из-за ограничений:
·     размер базы данных до 10GB.
·     Может использовать до 1GB памяти и 1го процессора (ядра), даже если установлено значительно больше.
 
DPA разработан с использованием стандарта ORM DataObjects.NET, поэтому работает с:
·       Oracle 10g, 11g, 12c, 18c;
·       Microsoft SQL Server CE 3.5, 4.0;
·       Windows Azure SQL Database 2008 R2, 2012, 2014, 2016;
·       Postgre SQL 8.3 - 11.2;
·       Firebird 2.5, 3.0;
·       MySQL 5.1, 5.5, 5.7, 6.0, 8.0;
·       SQLite 3 - 3.27.2.
 
Мы можем помочь в установке сервера баз данных. Направьте запрос в X-tensive.
