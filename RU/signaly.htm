Сигналы

 
Модуль сигналы предназначен для настройки логики отправки оповещений и журналирование фактов возникновения событий, заданных пользователем.
На панели навигации нажмите на кнопку , и в выпадающем списке выберите пункт «Сигналы».
Главное меню. Стартовая страница
Нажмите на кнопку в левом верхнем углу экранной формы, выберите пункт «Сигнал» из выпадающего списка.
Сигналы. Инициация добавление сигнала
В открывшемся модальном окне введите наименование сигнала. Для того чтобы указать рабочие центры, по данным поступающим от которых, необходимо сформировать сообщение - нажмите на кнопку .
Сигналы. Добавление сигнала
 
В модальном окне "Наблюдаемые подразделения и рабочие центры" выберите требуемые пункты меню и нажмите на кнопку . В поле "Выбранные подразделения и рабочие центры" отобразятся выбранные позиции. Для того чтобы удалить позиции из перечня, нажмите на кнопку .
Нажмите на кнопку , чтобы зафиксировать внесенные изменения.
Сигналы. Добавление сигнала. Наблюдаемые подразделения и рабочие центры
 
Задайте время группировки: в случае, если в течении заданного времени, созданный сигнал сработает несколько раз, пользователям будет направлено одно сообщение.
Укажите, необходимо ли активировать сигнал сейчас (активация может быть выполнена позднее).
Нажмите на кнопку , чтобы зафиксировать внесенные изменения в модальном окне "Создание сигнала".
Результат наполнения полей для создания сигнала
 
Сигнал может срабатывать по событию ("Триггер") и/или периодически ("Таймер").
Для описания условий формирования сигнала нажмите на кнопку на панели навигации, и выберите пункт «Триггер».
Сигналы. Инициация добавления триггера
 
Введите название триггера (для сигнала может быть создано несколько триггеров) и таймаут срабатывания (время, в течении которого события/состояния должны сохранять заданные значения).
Если триггер должен сработать при выполнении одного условия (логическое «ИЛИ»), в выпадающем списке выберите пункт «любое», если для сработки триггера необходимо, чтобы все условия были выполнены (логическое «И»), в выпадающем списке выберите пункт «все».
Создание триггера сигнала
 
Может быть задано одно или несколько условий срабатывания. Каждое из условий может представлять собой единичное правило или группу правил, для группы так же может быть задано правило проверки (логическое «И» / «ИЛИ»). 
Нажмите на кнопку , для того чтобы добавить правило. В появившихся полях, с помощью выпадающих списков, выберите необходимые для заполнения позиции.  
Триггер. Добавление правила
Нажмите на кнопку , для того чтобы добавить группу правил. Для построения правила может быть выбран любой из журналируемых параметров – создайте правило, последовательно используя выпадающие списки.
Триггер. Добавление группы правил
Нажмите на кнопку , чтобы зафиксировать внесенные изменения.
Результат добавления триггера
 
Для создания сигнала, срабатывающего периодически (в установленное время/с заданными интервалами) нажмите на кнопку на панели навигации, и выберите пункт "Таймер".
Сигналы. Инициация добавления таймера
 
Задайте точное время срабатывания таймера или укажите, с какой периодичностью он должен срабатывать, с помощью кнопки . После того, как все поля будут заполнены, нажмите на кнопку .
Сигналы. Добавления таймера
 
Результат добавления таймера
Для создания расписания активности сигнала, нажмите на кнопку и выберите пункт «Период». Или выполните клик правой кнопкой мыши на позиции «Расписание» и выберите пункт «Добавить/Период».
Сигналы. Создание расписания активного периода сигнала
Настройте расписание работы сигнала, нажмите на кнопку .
Результат создания расписания
Для описания реакции системы на факт срабатывания сигнала, нажмите на кнопку на панели навигации, и выберите пункт «Обработчик».
 Сигналы. Инициация создания обработчика
 
Укажите наименование обработчика (для сигнала может быть настроено произвольное количество обработчиков). Создайте описание, укажите тип обработчика и шаблон.
Нажмите на кнопку .
Создание обработчика. Веб-уведомление.
Укажите пользователей системы, которые должны быть проинформированы о факте срабатывания данного сигнала и нажмите на кнопку .
 
Обработчики. Выбор адресатов для обработчика
 
Примечание: Для отправки email уведомлений с вложениями, выставите флаг «Прикрепить отчеты». В открывшемся модальном окне с помощью выпадающих списков укажите условия формирования отчета.
Создание обработчика. Email уведомление.
Нажмите на кнопку , чтобы зафиксировать внесенные изменения.
Для активации сигнала выполните клик левой кнопкой мыши на заголовке сигнала и выставите флаг «Активно», нажмите на кнопку .
Активация сигнала
Чтобы отследить факты срабатывания триггеров в режиме реального времени, выберите пункт «Состояние». Нажмите на кнопку и укажите, мониторинг каких рабочих центров необходимо осуществлять.
Проверка состояния сигнала на оборудовании
Нажмите кнопку на выбрать , для того чтобы зафиксировать выбранные параметры.
 
    Текущее состояние сигнала
 
Чтобы отследить историю последних срабатываний сигнала, выберите пункт «Срабатывания». Для просмотра детальной информации, выберите строку в таблице срабатывания.
История срабатываний сигнала. Детальная информация
Для того чтобы настроить отображение уведомлений на панель диспетчера нажмите на кнопку на панели навигации, и в выпадающем списке выберите пункт меню «Настройки системы».
Инициация перехода в модуль "Настройки системы"
 
В настройках системы выберите пункт "Диспетчеризация".
Настройки системы. Диспетчеризация
 
В выпадающем списке откройте пункт меню "Сигнал". В поле "Подписаться на сигналы" в выпадающем списке выберите требуемый сигнал, который будет поступать на диспетчерский пульт. Если нужно указать несколько - воспользуйтесь кнопкой , а для удаления сигналов кнопкой .
Для того чтобы указать для каких групп будет доступен указанный сигнал, в поле "Пользователи и группы" укажите пользователей/группы пользователей, с помощью кнопок .
Нажмите на кнопку , чтобы зафиксировать внесенные изменения на экранной форме "Настройки системы".
