Enterprise structure

Now it is necessary to create a logical structure of the enterprise:
Create workshops and departments.
Distribute the added equipment.
Create working hours.
Configure accesses for employees to Structure.
Click , select «Enterprise structure».
 
In the left part of a screen hierarchical structure of the enterprise is displayed.
                                                                                                                                                                   Example
 
The system supports preset structure. You can describe in detail structure of an automation object using the following elements:
· Primary production
· Production site (workshop)
· Department (section/line)
· Equipment (working device)
 
Get up on any existing element of hierarchy and execute one of the following actions:
· Create new structural element
· Tie the connected equipment to structural element like Department.
                                                                                                                                  Create structure item
 
Set Name and description, click "Create".
                                                                                                                                                   Create structure Item.
Select the connected driver and click "Create".
                                                                                                                                               Add equipment object
 
Note: if during creation of the driver the step of creation of a structure member was not passed, repeated creation of the working center is not required, however it is possible to make a driver rebinding to again safe element (without data loss).
