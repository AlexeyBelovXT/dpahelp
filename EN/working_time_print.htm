Working time

 
For the enterprise level, the production site and department it is possible to set working hours.
Working time is inherited from parent to child.
You can disable inheritance on any item and specify a unique schedule.
For setup select the corresponding structure item, select "Schedule" tab.
 
                                                                                                                                                 Selected item of structure
 
You can manually adjust the schedule. We recommend that you create a template or select a pre-installed template.
To view templates click and select "Reference Books".
You need two items:
1) "Shift name" - to create shifts.
Two production shifts pre-installed. You can click to add another one.
 
2) "schedule templates" - to create template for working times.
Four templates pre-installed:
  • Work day, one shift. 08:00 - 12:00 and 13:00 - 17:00
  • Pre-holiday day, one shift. 08:00 - 12:00 and 13:00 - 16:00
  • Standart work week, one shift. From MO to FR. 08:00 - 12:00 and 13:00 - 17:00
  • Day off. Template to clear schedule.
You can change any template or create own. Select template and click for editing or Click to create new template.
1) Enter Name
2) Select type: day or week
3) drop shift box onto timeline and stretch it. Or click "More details" and specify the exact time.
4) Save template
 
Return to corresponded structure item. Click Schedule tab and click "Apply template"
Select template and date range. Click Apply
