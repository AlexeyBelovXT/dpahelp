Users settings

On HomePage click , and select "Security"
 
Select "Users" tab
                                                                                                                                                                  Security. Users tab
 
By default in a system one account is created: admin.
Click in the upper left corner to add user.
In the opened window enter user data, click  .
                                                                                                                                                                 Security. Create user
Click to edit properties of selected user.
                                                                                                                                                Security. Change user settings
 
Note: By default newly created users are added to the Users group.
To add or delete groups which included the user, select the user from the list and click . Click "Membership in Groups" tab to change user's membership.
Security. Membership in group tab
 
                                                                                                    Security. Change membership. Select group to add in.                                                                                         
 
On the Entry methods tab you can select a way of authorization. The method can be selected for this user if it was activated globally within a system.
Click to add method:
1) Login/password - DPA authorization
2) Windows - local computer or domain user
3) barcode or rfid. If barcode reader connected to computer, you may set this method.
                                                                                                                                      Security. Entry method tab
 
You can set the login and the password for the user. Also you can specify whether the user needs to change the password at the first input or upon the expiration of its action.

On the Contacts tab you can specify a contact information of the user. The entered data can be used for mailing of system messages (if type of entry = "working").
