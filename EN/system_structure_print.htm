System structure

DPA is the program complex including the following components:
· DPA Server – is responsible for data collection from the equipment. Carries out preprocessing and transfers information to DPA Host in real time;
· DPA Host – aggregates data from DPA Server(s). Writes the acquired information in a DB of DPA and files of DPA indicators for the further analysis. Is responsible for operation of the user interface;
· DPA DB – a relational database. Stores system tables, the help information, machine and user data;
· Realtime DPA DB - the database of real time. It is intended for optimization of write processes and reading machine data and also optimization of use of resources of the server.
· The driver – a set of program instructions for component interaction of a system with the CNC of a certain type of the equipment.
· The virtual driver is the Driver imitating operation of the machine.
· The agent - a program component. Provides collecting and data transmission about work and a status of the equipment to the server. Is installed directly on the machine. Preset tunings by X-tensive's specialists are required.
