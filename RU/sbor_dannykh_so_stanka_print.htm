Сбор данных со станка

 
Первым этапом настройки станка была установка сетевого подключения к Локальной сети.
Теперь надо определить способ получения информации со станка.
 
Есть два варианта:
1) с помощью STEP7
Если на станке включена поддержка STEP7, открыт порт, то DPA может получать данные со станка напрямую, без установки специального ПО.
Т.е. для подключения драйвера достаточно знать IP адрес сетевого интерфейса, через который будет передаваться информация.
Примечание: косвенным подтверждением поддержки STEP7 является открытие порта через HMI, в настройках брандмауэра.
 
2) с помощью агента DPA
Если на станке нет доступа к функционалу STEP7, то необходимо устанавливать агент DPA.
Примечание: Если на PCU станка есть три сервиса ncdde, mbdde, dhserver. Это службы, которые в виде подписки скидывают агенту все изменения на индикаторах станка. Т.е. оборудование поддерживает установку агента.
