Какие функции доступны через REST API
 
Полный перечень функций, которые доступны через REST API DPA можно получить, открыв в браузере ссылку:
http://localhost/swagger
Вместо “localhost” укажите имя сервера, где развернут DPA.

Все функции, реализованные в DPA, и доступные через web-интерфейс пользователям, так же доступны через REST API. Другими словами, REST API позволяет использовать все функциональные возможности DPA.
Есть ли документация на REST API
REST API пока не имеет отдельной документации. Мы специально давали методам интуитивно понятные названия, организовали информацию в виде структурированного списка.
Настроили систему, которая дает информацию об аргументах методов, а так же поможет выполнить тестирование, вызвать методы, получить ответ в JSON формате.
 
Безопасность
REST API доступен только авторизованным пользователям.
При обращении к функциям REST API происходят все те же проверки прав и привилегий, которые выполняются при обычной работе в системе под учетной записью соответствующего пользователя.
Авторизация происходит путем вызова метода /api/Account/Login.
