Import Enterprise Structure

 
To make import of equipment it is necessary to select the corresponding item in a tree "Structure of the enterprise", right-click on it and select "Import".
Enterprise structure
 
Click checkbox "Import settings of the driver" to display additional settings.
Enterprise structure. Import. Step 1
After additional settings were displayed, select transport and click "Next"..
Enterprise structure. Import. Step 2
In the opened dialog box it is necessary to select the corresponding configuration of the driver and click "Next" .
Enterprise structure. Import. Step 3
Click Import to apply settings .
Enterprise structure. Import. Step 4
 
Notifying windows about successful import of a configuration of the equipment will appear.
Enterprise structure. Import. Step 5
 
Note: Repeat Import If only two notifying windows appeared.
Enterprise structure. Import. Notifications
 
Note: It is also necessary to check Event and Field fields, in the Indicators. Repeat Import if empty.
