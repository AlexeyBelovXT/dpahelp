Систему можно развернуть и на SQL Express. Но не рекомендуем развёртывание осуществлять на SQL Express для промышленной эксплуатации DPA, так как возможны проблемы с производительностью, и существуют ограничения по размерам БД.
 
DPA разработан с использованием ORM DataObjects.NET, поэтому кроме указанных СУБД, поддерживает:
  •   Oracle 10g, 11g, 12c, 18c;
  •   Microsoft SQL Server CE 3.5, 4.0;
  •   Windows Azure SQL Database 2008 R2, 2012, 2014, 2016;
  •   Postgre SQL 8.3 - 11.2;
  •   Firebird 2.5, 3.0;
  •   MySQL 5.1, 5.5, 5.7, 6.0, 8.0;
  •   SQLite 3 - 3.27.2.
 
Чтобы развернуть DPA на одной из перечисленных СУБД, направьте запрос в X-tensive.
