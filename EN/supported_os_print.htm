Supported OS

 
The DPA components are installed on x64 versions of the following MS Windows operating systems:
· Windows Server 2016 Standard
· Windows Server 2016 Datacenter
· Windows Server 2012 R2 Standard
· Windows Server 2012 R2 Datacenter
· Windows Server 2008 R2 Standard
· Windows Server 2008 R2 Datacenter
· Windows 10
· Windows 7
 
Also NET Framework 4.6.2 is required (https://www.microsoft.com/ru-ru/download/details.aspx?id=53344).
