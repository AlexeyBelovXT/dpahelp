Installation options

 
It is necessary to consider:
  • The number of the connected equipment - more equipments, more data, more servers for DPA;
  • Geographical distribution - the workshops which are geographically divided from each other will require separate DPA-servers;
  • Security policy - all components it is favorable to place on one powerful server, but user access to network of machines increases risks.
     
    The main system characteristics of the server hardware:
  • Memory capacity;
  • Processor capacity;
  • Speed of a read and write of harddisks on the databases server;
