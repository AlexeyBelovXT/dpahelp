Безопасность

 
Для уровня предприятия, производственной площадки, подразделения и рабочего центра могут быть заданы настройки доступа. По умолчанию у всех пользователей есть доступ ко всем элементам структуры.
Для настройки выберите соответствующий элемент структуры, на вкладке «Безопасность», установите флажок «Только указанным группам» и нажмите на кнопку .
                                                                                                                              Настройки безопасности элемента структуры предприятия. По умолчанию
Выберите группы доступа, которым разрешено просматривать данные мониторинга по выбранному оборудованию (группе оборудования), нажмите на кнопку .
