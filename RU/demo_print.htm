Для проектов с небольшим количеством станков и пилотных проектов допустимо устанавливать все компоненты на один сервер.
При установке всех компонентов системы на один сервер (подключение до 10 единиц оборудования), необходим следующий набор компонентов, с минимальными требованиями:
  • Операционная система: Windows Server 2016 Standard x64;
  • СУБД: Microsoft SQL Server;
  • Оперативная память: 8 Гб;
  • Жесткий диск:  480Гб;
  • Ethernet  – 100Мбит.
