Установка

 
В зависимости от выбранного варианта развертывания, вам потребуется провести одну или несколько сессий установки системы. Воспользуйтесь инструкцией по работе с единым установщиком, если DPA Host и DPA Server будут размещены на одной машине.
 
Если DPA Server вынесен на отдельную машину, вам понадобится раздел Установка DPA  Server.
