Installation files

In the special Section you can download installation files for deploying of Agents on certain types of machines.
On a navigation bar click , and select "Installation files".
On the screen form "Installation files" select the file which needs to be downloaded for further software installation.
For downloading the file, double-click on necessary point, or select agent and click in the left part of a screen.
 
The system will load the file with extension ".msi" into Downloads folder.
