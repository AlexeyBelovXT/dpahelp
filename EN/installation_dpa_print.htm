Installation DPA

Download the current version of the installer from our website. Start the Setup.DPA.(version).exe.
1. Select language from the drop-down list which will be used in installation process.
                                                                       1. Installation wizard. Start
Click "Next".
2. Carefully read license agreement.
If you agree with conditions, click "Accept". For end of installation click "Cancel".
                                                       2. Installation wizard. license agreement
 
3. By default offered to install all DPA components in standard folders.
If it is necessary to set only one of a component, remove an excess tick.
Click if you whant to select other folder for installation.
Note: Do not change the parameters set by default needlessly.
 Click "Next" .
                                                            3.  Installation wizard. Choosing of components 
 
4. The Wizard requests on behalf of what user the service DPA Host should be launched.
 4.1 Installation wizard. DPA Host
 
Also data for connection to the database are requested: instance name, a name of the existing or newly created database.
By Services console on the server where MS SQL is set it is possible to learn a name of instance.
 4.2 The instance name of the database is entered in brackets near a name of service SQL Server
 
By default the system suggests to create the DPA database on behalf of user which started installation. You can specify other way of the authorization/representative of the user and also select the existing database or create new. To do this click button .
 4.3 Installation wizard. Install DPA Host. Database options
Click "Create" , to fix settings. Click "Next" .
 
5.On this step the system confirms the selected settings. You can make changes and apply settings, having clicked "Setup of the Database".
                                                       5. Installation wizard. Database confirmation
 
6. The system supports the simplest preset structure.
Set checkbox opposite to those elements which will need to be added to structure of an automation object:
 Add virtual workshop (the enterprise, the production site (workshop), division (section/line) and the equipment (the working center) will be created, the reference book "equipment 3D models" will also be completed and the standard symbolic circuit is added);
 Add a product, the managing program and technology (reference books of "Product", "Product Type", "Managing Programs" and "Type unitary enterprise" will be completed and also the technology of a product is added;
 Add users and group (minimum necessary edited list of groups and users of a system will be added);
Note: Miss this step if you install DPA in production. Not for testing. It is enough to click "Next" without any checkbox selected.
6. Installation wizard. pre-installed objects
 
7. The system requests on behalf of what user the service DPA Server should be launched.
Note: It is necessary to be convinced that the selected user has necessary rights.
Select necessary settings and click "Next". .
                                                        7. Installation wizard. Options for DPA Server
8. СThe system requests confirmation of parameters of the unit.
Note. If you doubt correctness of the choice of settings, consult at X-tensive.
If all data are right, click "Install" .
                                                     8. Installation wizard. Confirmation
 
9. Wait for ending the installation.
 
                                                                      9. Installation wizard. Installation
 
10. congratulation, the system is successfully installed.
                                                            10. Installation wizard. Installation completed
