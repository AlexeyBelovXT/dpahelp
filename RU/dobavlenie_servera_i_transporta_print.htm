Добавление сервера и транспорта

 
На панели навигации нажмите на кнопку , и из выпадающего списка выберите пункт «Мониторинг».
 
Добавляем DPA Server
1. На панели навигации нажмите на кнопку и выберите пункт "Сервер".
 
Укажите имя сервера, его адрес и порт. По умолчанию сервер использует порт «45617».
Добавление сервера. Шаг 1
 
Проверить актуальность значения можно с помощью командной строки, запущенной на DPA Server от имени Администратора: «netstat -aon | more» и PID DpaServer.
В данном случае DPA Server запущен на том же компьютере, что и DPA Host
                                                                                                               
2. Нажмите на кнопку .
