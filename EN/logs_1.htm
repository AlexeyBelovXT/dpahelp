Logs

Select server and open the Logs tab. Specify the log (if is configured a little), which needs to be downloaded, and click .
log-file download
Note: Also files can be copied manually in the file system (%ProgramData\X-tensive\DPA\Logging).
example 
 
DPA server supports several modes of logging:
• Off - logging is disabled;
• Fatal – Critical errors leading to failure of the DPA server;
• Error – Logging of all errors: errors of drivers, calculations of statuses, unsuccessful sending events;
• Warn – Logs emergency situations which can be processed; 
• Info – logs start of services, external requests, messages of drivers and all higher levels (Warn, Error, Fatal);
• Debug – Journalizing in a debug mode: statistics of traffic, detail of work of functions (it is not recommended for regular use);
• All - Full journalizing of messages of all levels (it is not recommended for regular use).
 
The Info mode is by default exposed (this log cannot be deleted).
Enter a name of the new log and level of logging, and click "Add".
