Update DPA

 
1. Download and start new application Setup.DPA.(version).exe. Make sure that checkbox is selected.
1. Update wizard. Beginning
 
Click "Update" .
Note: if you remove "Save Settings" checkbox, the system will be installed again.
2. Wait several minutes.
2. Update wizard.
 
3. Congratulation. The system is successfully updated. Click "Finish" .
3. Update wizard. Finishing
