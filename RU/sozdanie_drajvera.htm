Создание драйвера

 
Выберите DPA Server, к которому необходимо подключить оборудование. Нажмите на кнопку на панели навигации в левом верхнем углу экранной формы:
                                                                                                                                              Инициация создания драйвера через панель навигации
 
Возможные варианты подключения:
                                                                                                                                                                      Варианты создания драйвера
 
1. При выборе пункта «Вручную», необходимо указать все параметры подключения (вам может потребоваться помощь специалистов X-tensive).
Данный пункт так же применяется для подключения виртуальных устройств (не существующее оборудование. активность имитируется с помощью встроенной на DPA Hostпрограммы).
2. «Ethernet» - автоматический поиск и настройка драйвера по IP;
3. «Serial» позволяет создать драйвер для специфичных ЧПУ, подключаемых через COM-порт (не сетевое подключение);
4. «Импорт» применим в случаях, когда требуемый драйвер не предустановлен в системе (вам потребуется помощь специалистов X-tensive);
 
Рекомендуемый для пользователей способ - «Ethernet».
Если у вас есть информация о производителе и типе ЧПУ укажите их для ускорения поиска, укажите IP и нажмите на кнопку  . 
                                                                                                                                         Создание драйвера через Ethernet. Шаг 11
 
Ожидайте. Система запрашивает параметры единицы оборудования по указанному IP-адресу.
                                                                                                                                              Создание драйвера через Ethernet. Шаг 2
Система обнаружила драйвер на указанном IP-адресе. Нажмите на кнопку .
                                                                                                                                                      Создание драйвера через Ethernet. Шаг 3
 
Для загрузки конфигурации драйвера (настроек чтения параметров) выберите пункт «Из существующих», и в выпадающем списке, выберите необходимую конфигурацию драйвера.
                                                                                                                                                Создание драйвера через Ethernet. Шаг 4
Выберите требуемый транспорт, установите флаг и нажмите на кнопку .
                                                                                                                                              Создание драйвера через Ethernet. Шаг 5
 
Введите наименование оборудования. По умолчанию в системе описан простейший план предприятия, оборудование должно быть закреплено за конкретным подразделением. В последующем, после конфигурации структуры предприятия, выбор может быть изменен. 
                                                                                                                                                    Создание драйвера через Ethernet. Шаг 6
Подтвердите создание драйвера, нажав на кнопку .
                                                                                                                                                  Создание драйвера через Ethernet. Шаг 72
Если у вас нет точной информация о производителе и типе ЧПУ , укажите IP и нажмите на кнопку  .
                                                                                                                                                      Создание драйвера через Ethernet. Шаг 1
 
Ожидайте. Система запрашивает параметры единицы оборудования по указанному IP-адресу.
                                                                                                                                                   Создание драйвера через Ethernet. Шаг 2
Остальные пункты формирования драйвера в автоматическом режиме аналогичны пунктам, приведенным выше, для формирования драйвера в предзаполненном режиме.
 
К одному DPA-серверу может быть подключено несколько станков. Определение оптимального количества подключаемого оборудования описано в рекомендации по установке системы.
В случае подключения оборудования через COM-порт, перечень указываемых параметров специфичен и включает параметры COM-портов ЧПУ и сервера.
 
Пример заполнения и источник данных (BaudRate – бит в секунду, Parity - четность, DataBits - биты данных, StopBits – стоповые биты, Hadshakes – управление потоком):
                                                                                                                    Доступ к настройкам идет через Панель Управления - Диспетчер устройств
 
Минимальный таймаут чтения 20 миллисекунд, таймаут по умолчанию – 200 миллисекунд.
