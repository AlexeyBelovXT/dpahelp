All-in-one

For projects with the number of machines to five and test-projects it is admissible to install components on one server.
The recommended system requirements:
  • Windows Server 2016 Standard
  • MS SQL Server 2017 Standard x64
  • Processor: frequency is 3 GHz, 16 cores
  • Memory: 32GB
  • Harddisks for OS and the DPA components: SAS 10K > 120GB .
  • Harddisks for DPA database: SSD > 200GB.
  • Network connection: 1gb/s
     
 
Notes.
We recommend to integrate disks for OS in RAID1 (2 disks). RAID1 capacity = a half from the general capacity of disks. Such array will increase fault tolerance of a disk system.
We recommend to integrate disks for Database RAID10 (even quantity, 4 disks and more). RAID10 capacity = a half from the general capacity of disks. Such array will provide the maximum capacity and fault tolerance of a disk system.
