Licensing

 
For data collection from the equipment the active license is needed. Pay attention that in the presence of several DPA servers the corresponding number of license files will be required.  
Click , select Monitoring. Select server and click License. Press "Create request".
                                                                                                                                                                       Step 1
 
In the opened window select license type: Production, Trial or Trial No Hid. Last two for testing only.
 
If the license "Production" is selected, it is necessary to specify the number of worked centers (cnc machines). In the field "Is program Loader?" set a checkbox. It's necessary to manage control programs on this server.
Click "Upload".
                                                                                                                                                 Step 3
Note: Production license is termless.
Trial license works up to 30 days.
Trial No Hid license of this type creates the identifier on the basis of data on your equipment which does not allow to use this license on other equipment. It is valid up to 30 days.
 
The system will generate the file with extension ".xml" and will save it in the Downloads folder.
 
Send the generated file to X-tensive OR to System integrator.
Receive in reply the file with license.
Load the received file by clicking "Install License".
 
Now it is necessary to create users, groups, enterprise structure and add equipment.
