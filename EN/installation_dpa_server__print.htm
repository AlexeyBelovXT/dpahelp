Installation DPA Server

To download the installer for DPA Server: on the homepage click and from the drop-down list select the "Installation files".
Main menu. Home page 
On the screen form "Installation files" select the DPA Server item from the submitted list, and press the button . File Setup.DPA.Server.(version).exe will be saved in the specified directory.
Installation files menu. 
 
1. Start application Setup.DPA.Server.(version).exe.
                                        1. Installation wizard. Start
 
Click "Nezt".
2. Attentively read license agreement. If you agree with conditions, select checkbox and click "Next", else click "Cancel" .
                       2. Installation wizard. license agreement
 
3. Check necessary requirements to an empty space on a disk. If it is necessary change an installation directory.
Note: Do not change the parameters set by default needlessly.
 
Нажмите кнопку .
       
                                  3.1 Installation wizard. Free space on drive
 
                             3.2 Installation wizard. Choose installation directory
 
4. The system requests on behalf of what user the service DPA Server should be launched.
Note: It is necessary to be convinced that the selected user has necessary rights.
 
Нажмите кнопку .
                             4.  Installation wizard. user selection
 
5. The system requests confirmation of parameters. Click "Install" .
                           5. Installation wizard. Confirmation
 
6. Wait several minutes.
                                           6. Installation wizard. Installation started
 
7. Congratulation. The system is successfully installed.
                            7. Installation wizard. Finishing
 
Click "Finish" , for completion of installation DPA Server.
