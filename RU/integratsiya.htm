Интеграция

 
DPA поддерживает синхронизацию справочника "Управляющие программы". Существует несколько способов синхронизации справочников между несколькими системами:
  • через вызов WEB-сервисов;
  • очередь MSMQ - это надежный вариант, гарантирующий доставку сообщений.
  • обмен XML-файлами через общий локальный или сетевой каталог (каждый XML-файл содержит определенный запрос, а вызывающая система помещает "файл-запрос" в общий каталог, после  чего "система-получатель" периодически просматривает этот общий каталог и обрабатывает поступившие XML-запросы).
 
В настройках DPA можно активировать синхронизацию и выбрать общий каталог для обмена файлами.
Данная ссылка на FTP-сервер, где представлена XSD-схема, которая описывает формат обмена XML-файлами: ftp://ftp.x-tensive.com/Public/Downloads/DPA.Schemas/Xtensive.DPA.Tasks.xsd
 
In
В данный каталог должны поступать XML-файлы с запросами
Done
DPA перенесет те XML-файлы в каталог, которые были успешно обработаны
Error
В подкаталог попадут XML-файлы, при обработке которых возникла ошибка.
Примечание: Описание ошибки добавляется в конец XML-файла в формате "<error>Текст ошибки</error>".
Out
Каталог для исходящих запросов.
 
Система поддерживает следующие типы запросов:
  • Create;
  • CreateOrUpdate;
  • Update;
  • Remove.
