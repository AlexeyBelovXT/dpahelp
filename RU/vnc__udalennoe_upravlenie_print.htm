VNC. Удаленное управление

 
 
Начиная с версии ??? в DPA появилась функция подключения к станку через систему удаленного доступа VNC (Virtual Network Computing).
Обычно, для удаленного подключения устанавливается специальная программа (VNC viewer).
Мы встроили такую программу прямо в систему DPA.
 
Для настройки подключения к станкам необходимо изменить конфигурационный файл.
Перед этим вы должны узнать значение сетевых настроек подключения, которые будете вносить в конфигурацию.
 
По каждому станку у вас должна быть следующая информация:
·     IP-адрес станка. Этот адрес можно узнать через DPA host, в настройках Драйвера.
Открыли Структуру Предприятия, выбрали станок
кликнули на драйвер
Записали IP-адрес.
 
·     Порт подключения к VNC сервису станка. Обычно это 5900 (или 5800), но лучше посмотреть настройки непосредственно на станке.
·     Пароль на просмотр изображения со станка
·     Пароль на управление станком
Примечание: Пароли могут быть пустыми («»). Все настройки необходимо сверять на станке.
 
 
Теперь можно приступать к редактированию файла, который располагается на сервере с установленным DPA Host
 
1)     Открываем папку “C:\ProgramData\X-tensive\DPA VNC\”
Примечание: по-умолчанию папка C:\ProgramData является скрытой, поэтому вы не увидите ее через Проводник. Рекомендуем в адресной строке сразу указать полный путь до папки и нажать Enter
Либо выбрать Пуск-> Выполнить (комбинация клавиш WIN+R) и указать путь до папки, после чего нажать Enter
 
2)     В папке редактируем файл DpaVnc.config
В этом файле содержатся настройки станков, к которым можно подключаться по VNC
Настройка каждого отдельного станка начинается с открывающей фигурной скобочки { и заканчивается закрывающей фигурной скобочкой }
Примечание: Название и значение параметра указываются в двойных кавычках
 
Внутри скобочек необходимо указать следующие параметры.
 
  • «name»: это наименование станка
  • «url»: адрес подключения, указывается в формате «IP-адрес/порт»
  • «view»: пароль для просмотра экрана станка.
  • «control»: пароль для удаленного управления станком
    Примечание: Если пароля нет, то пустое значение указывается кавычками без символов внутри:   «»
 
 
Пример конфигурации одного станка:
 
{
“name”: “Станок1”
“url”: “10.0.0.183/5900”
“view”: “”
“control”: “@dm1n”
}
 
Примечание: если необходимо отключить удаленное управление, то вместо удаления параметра достаточно в начале строки поставить символы комментария //
Например:
//“control”: “@dm1n”
Тогда параметр не будет учитываться при организации подключения.
 
3)     Сохраняем файл
4)     Переходим по адресу http://имя_DPA_host:9000, вместо имя_DPA_host укажите полное имя или IP-адрес сервера, на котором установлен DPA Host.
Появится примерно такая страница:
Это список станков и ссылки с Разрешенными действиями, к которым был указан пароль.
Если были разрешены как Просмотр, так и Управление, то в строке будут две ссылки "View" и "Control".
