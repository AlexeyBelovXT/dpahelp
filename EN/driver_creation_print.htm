Driver Creation

 
Select DPA Server to which it is necessary to connect the equipment. Press the button and select Driver
Select Driver
 
Possible options:
1. When choosing the "Manually" option, it is necessary to specify all parameters of connection independently, without hints of a system.
This option is also applied to connection of virtual devices (not existing equipment. the activity is imitated by built-in program).
2. "Ethernet" - automatic search and setup of the driver on IP;
3. "Serial" allows to create the driver for specific the CNC, connected through COM port (not network connection);
4. "Import" is applicable in cases when the required driver is not preset in a system. You will need the help of specialists X-tensive;
 
Note. A large number of Drivers are connected through "Ethernet".
If you have information on the vendor and the CNC type specify them for search acceleration, specify IP and click "Next".

Wait. The system requests parameters to the specified IP address.

If the system did not find the driver, check network connection to the machine. If the driver is found, on the specified IP address, click "Next" .

For loading of a configuration of the driver select "From existing", and choose a necessary configuration from the list.

Select required transport. Click "Next" .
