Настройка интеграции

 
Система предоставляет возможность интеграции со смежными системами посредством web-api и xml-файлов. 
Посредством web-api могут быть импортированы и экспортированы любые данные доступны в графическом интерфейсе пользователя. 
 
Примечание: Для применения в рамках проекта web-api потребуется помощь специалистов внедрения.
 
Самым простоим способом для освоения интеграции является обмен xml-файлами. 
На панели навигации нажмите на кнопку , и из выпадающего списка выберите пункт «Настройки системы».
Главное меню. Стартовая страница
 
Посредством xml-обмена можно наполнить справочники системы:
  •  Загрузить эталонные тексты управляющих программ;
  •  Указать перечень производимой продукции;
  •  Дать описание технологии их изготовления.
Мониторинг. Настройки системы. Интеграция
 
Помимо справочных данных существует возможность загрузить периоды планового обслуживания и сменно-суточные задания. Для этого необходимо включить опцию «Импорт» и указать путь к выбранной директории размещения xml-файлов. Нажмите на кнопку , для того, чтобы сохранить внесенные изменения.

Вместе с системой поставляются xsd-схемы, позволяющие службам, сопровождающим уже внедренные на предприятии системы быстро освоить формат выгрузки данных.
Пример отображение xsd-схемы
 
Также существует возможность передать данные в формате xml (фактическую длительность выполнения операций, производство по задачам, длительность, причины простоев оборудования, результаты процедур обслуживания станков) в смежные системы. Для этого укажите путь директории куда будут помещены файлы, проставите флаги в необходимых полях и нажмите на кнопку .
