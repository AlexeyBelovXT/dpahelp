Adding DPA server and transport

 
On a navigation bar press the button , and from the drop-down list select the Monitoring item.
 
Adding DPA Server
1. On a navigation bar press the button  and select "Server".
 
Enter server name, its address and port. By default the server uses 45617 port.
Step 1
 
It is possible to check relevance of value by command line started on DPA Server on behalf of the Administrator: «netstat -aon | more» and PID DpaServer.
In this case DPA Server is started on the same computer, as DPA Host
                                                                                                               
2. Click "Next" .
                                                                                                                                                         Step 2 The system checked that the server is available.
3. Select transport which needs to be added. It is recommended to select WCF transport as less resource-intensive. Click "Next".
                                                                                                                                                                      Step 3
Click "Next" to add transport .
Click "Create" .
"No data" because no equipment is added.
 
The system will inform you on successful of operation.
Now we will add the license.
