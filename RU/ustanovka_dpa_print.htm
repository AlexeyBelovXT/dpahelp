Установка DPA

Запустите приложение Setup.DPA.(версия).exe.
Выберите язык из выпадающего списка, который будет использоваться в процессе установки.
                                                                       Мастер установки. Начало
Нажмите на кнопку .
Внимательно прочитайте условия лицензионного соглашения. Если Вы согласны с условиями, нажмите кнопку , в противном случае – кнопку .
                                                       Мастер установки. Лицензионное соглашение
 
Отметьте необходимые для установки компоненты, и укажите директорию установки.
Примечание: Не меняйте параметры, установленные по умолчанию, без необходимости.
 
Нажмите кнопку .
                                                            Мастер установки. Выбор компонентов 
 
Система запрашивает от имени какого пользователя должен производиться запуск службы DPA Host.
Примечание: Необходимо убедиться, что выбранный пользователь обладает необходимыми полномочиями.
Система запрашивает данные для подключения к базе данных: имя экземпляра, имя существующей или вновь создаваемой базы данных. Имя экземпляра базы данных Вы можете узнать, запустив установленную среду управления. По умолчанию система предлагает создать базу данных DPA от имени пользователя, чья учетная запись использована для установки системы. Вы можете указать другой способ авторизации/уполномоченного пользователя, а также выбрать существующую базу данных или создать новую, с помощью кнопки .
 Мастер установки. Конфигурация DPA Host
Нажмите на кнопку , для того чтобы зафиксировать выбранные параметры.
 Мастер установки. Конфигурация DPA Host. Выбор базы данных
Нажмите на кнопку .
                                                          Мастер установки. Конфигурация DPA Host
 
По умолчанию система предлагает настроить базу данных DPA от имени пользователя, чья учетная запись использована для установки системы.
Вы можете другой способ авторизации/уполномоченного пользователя.
Примечание: Необходимо убедиться, что выбранный пользователь обладает необходимыми полномочиями.
Нажмите кнопку .
                                                       Мастер установки. Настройка базы данных
 
Система содержит простейшую предустановленную структуру.
Проставьте флаги напротив тех элементов, которые необходимо будет добавить в структуру объекта автоматизации:
  •  Добавить виртуальный цех (будет создано предприятие, производственная площадка (цех), подразделение (участок/линия) и оборудование (рабочий центр), также будет заполнен справочник "3D-модели оборудования" и добавлена стандартная мнемосхема);
  •  Добавить изделие, управляющую программу и технологию (будут заполнены справочники "Изделия", "Тип изделия", "Управляющие программы" и "Тип УП", а также добавлена технология изделия;
  •  Добавить пользователей и группы (будет добавлен минимально необходимый редактируемый перечень групп и пользователей системы);
Мастер установки. Предустановленные данные
 
Система запрашивает от имени какого пользователя должен производиться запуск службы DPA Server.
Примечание: Необходимо убедиться, что выбранный пользователь обладает необходимыми полномочиями.
Нажмите кнопку . .
                                                        Мастер установки. Конфигурация DPA Server
Система запрашивает подтверждение параметров установки. Нажмите кнопку .
                                                     Мастер установки. Подтверждение установки
 
Ожидайте, идет установка.
 
                                                                      Мастер установки. Установка
 
Поздравляем, система успешно установлена.
                                                            Мастер установки. Завершение установки
