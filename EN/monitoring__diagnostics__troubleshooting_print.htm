Monitoring. Diagnostics. Troubleshooting

 
At emergency situation be convinced of the following:
· The license is active (The license can be deactivated at change of hid);
· All components are started;
· Versions of components are compatible;
· Data transmission (transport) is carried out to the correct IP address;
· The driver is included and available.
 
At regular system operation on the server should be services Dpa Host and DPA Server are started. If components are stopped, start them and direct a log file to X-tensive.
