Сеть и маршрутизация

Требования к сети:
 
В случае, когда DPA Server и DPA Host установлены на разных серверах:
  • со стороны DPA Host необходимо открыть порт TCP 45617;
  • со стороны DPA Server необходимо открыть порты TCP 45618 и UDP 1000-10000.    
 
Примечание: для того чтобы открыть требуемые порты, необходимо в стандартном Брандмауэре Windows (Windows Firewall) прописать правила для входящих/исходящих подключений (внести порты в список разрешенных).
