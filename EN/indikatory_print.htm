Индикаторы

Параметры мониторинга могут храниться в БД SQL и в базе данных реального времени.
По умолчанию, для предотвращения разрастания базы данных и снижения нагрузки на сервер, часто изменяемые параметры, такие как скорости нагрузки хранятся в базе данных реального времени.
На панели навигации нажмите на кнопку , и из выпадающего списка выберите пункт «Структура предприятия».
Главное меню. Стартовая страница
 
Чтобы проверить/изменить эту настройку перейдете в модуль «Структура предприятия», выберите единицу оборудования, раскройте индикаторы.
                                                                                                                                      Пример настройки хранения индикатора в файловой системе
Для того чтобы добавить индикатор, нажмите на кнопку в левом углу экранной формы, либо нажмите правой кнопкой мыши на пункт «Индикаторы», далее «Добавить».
                                                                                                                                          Инициация создания индикатора через панель навигации                 
 
                                                                                                                                                    Инициация создания индикатора через контекстное меню
В модальном окне «Создание индикатора оборудования» заполните все необходимые поля и нажмите на кнопку .
                                                                                                                                                                   Создания индикатора оборудования
 
Примечание: для того, чтобы выбрать необходимый формат хранения «В памяти», «В БД», «В файловой системе» - установите флаг в нужном пункте меню «История», и заполните поля при необходимости.
Нажмите на кнопку , для подтверждения введенных параметров.
