Network and Routing

Requirements to network:
  • The server or the PC on which the DPA component is installed should be connected to network via the network interface card with a speed 1gb/s.
  • If several DPA servers are required, We recommend to install the network interface card on DPA Host with a speed 10gb/s.
 
If DPA Server and DPA Host are installed on different machines, then:
  • from DPA Host it is necessary to open TCP 45617 port;
  • from DPA Server it is necessary to open TCP 45618 and UDP 1000-10000 ports.   
 
Note:
During installation the DPA component rules for standard windows firewalls are automatically created.
If you use another software for protection of network connections, address the system administrator.
Ask the network administrator to check that machines with the DPA components can set network connection to the equipment (machines with CNC).
