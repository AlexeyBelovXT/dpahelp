Sinumerik Operate 808D

 
Для смены режима работы надо указать соответствующий пароль (SUNRISE по умолчанию).
 
1) На панели кнопок нажимаем комбинацию
2) Выбираем раздел "Set password"
3) Указываем пароль:  SUNRISE
 
над кнопками меню должна появиться надпись нового режима работы
 
Внимание: после внесения изменений не забудь перейти в режим Оператора. Для этого на шаге 3) надо указать пароль OPERATOR (по умолчанию). Либо перезапустить контроллер.
 
Теперь можно посмотреть свойства системы и настроить сетевой интерфейс.
 
Как посмотреть свойства системы.
 
нажмите служебную клавишу , затем , далее
На экране будет отображена версия HMI, так же будет отображена версия операционной системы
 
Изменить сетевые настройки
 
После успешной смены режима работы, в горизонтальном наборе кнопок появляется "Диспетчер служб"
С помощью этой кнопки можно запускать утилиты по изменению параметров системы.
1) открываем "Диспетчер служб"
2) открываем "Управление сервисом"
 
Для изменения IP-адреса
3) открываем в вертикальном меню "Сетевой сервис"
здесь можно настроить сетевой адрес для интерфейса, подключенного к локальной сети предприятия.
По умолчанию адрес назначает DHCP. Можно изменить на статический.
Внимание: сетевые настройки необходимо уточнить у сетевого администратора Предприятия.
 
Для изменения настроек брандмауэра
3) открываем в вертикальном меню "Сервис брандмауэра"
По умолчанию необходимые порты удаленного управления закрыты. Открываем порты соответствующими галочками
22 - разрешить доступ по SSH (для удаленного администрирования операционной системой на станке). Не обязательно для DPA
102 - разрешить доступ к протоколу STEP7 (для автоматического сбора статистики). Обязательно для DPA
