Библиотека УП

 
Библиотека управляющих программ - это программный модуль, который служит для ведения хранилищ управляющих программ для станков с ЧПУ, обеспечивая базисные функции манипуляций с УП:
  • Скачивание и хранение управляющих программ;
  • Поддержка управления версиями управляющих программ;
  • Возможность указывать для производства каких изделий и на каких рабочих центрах может быть использована управляющая программа;
  • Загрузка управляющих программ на станок (при этом система контролирует состояние станка и не позволяет прерывать выполняющую программу).
 
Также "Библиотека УП" служит удобным средством для решения следующих задач:
  • Ведение перечня и параметров изделий, обрабатываемых на предприятии с применением УЧПУ;
  • Ведение комплектов и параметров УП для конкретных изделий;
  • Идентификация файлов УП;
  • Поиск УП и/или их отбор по фильтру наименования;
  • Контроль состояния актуальности УП;
  • Перемещение УП между следующими хранилищами: серверный компьютер, клиентские компьютеры, внутренняя память отдельных УЧПУ;
  • Контроль истории изменения файлов УП;
  • Просмотр, редактирование и модуляризация (разбиение на подпрограммы) файлов УП;
  • Контроль отличий версий УП.
 
Библиотека управляющих программ в сочетании с комплексом программных и аппаратных средств системы мониторинга создает условия для эффективной и безошибочной работы с множеством программ для УЧПУ.
