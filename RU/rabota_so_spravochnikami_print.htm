Работа со справочниками

 
Справочник «Управляющие программы» - централизованное версинное хранилище управляющих программ и связанных с ними нормативов.
Для того чтобы наполнить данный справочник - на панели навигации нажмите на кнопку , и в выпадающем списке выберите пункт «Справочники».
Главное меню. Стартовая страница
 
В открывшейся экранной форме существует несколько вариантов перехода к выбранному справочнику:
  • Выберите справочник «Типы причин простоя» и нажмите на кнопку   в левом углу экранной формы;
  • Выполните клик правой кнопкой мыши и выберите пункт «Перейти»;
  • Выполните двойной клик левой кнопкой мыши.
Справочники
 
 Аналогичным образом, через контекстное меню или панель навигации могут быть вызваны следующие операции:
- Добавить;
- Редактировать;
- Удалить;
- Перейти;
- Обновить.
На панели навигации нажмите на кнопку , для того чтобы добавить управляющую программу.
Справочники. Управляющие программы
 
В открывшемся модальном окне заполните все необходимые поля. Кнопка вызывает диалоговое окно, с возможностью выбора требуемых параметров из представленного перечня.
Справочники. Управляющие программы. Создание записи в справочнике
 
Нажмите на кнопку , чтобы зафиксировать указанные данные.
Справочники. Управляющие программы. Создание записи в справочнике. Выбор объекта
 
Для того чтобы зафиксировать введенные данные нажмите на кнопку .
Справочники. Управляющие программы. Создание записи в справочнике.
 
Примечание: Аналогичным образом можно отредактировать любую сохраненную управляющую программу, с помощью кнопки на панели навигации.
 
Для того чтобы удалить сохраненную управляющую программу из перечня, выберите требуемый пункт и нажмите на кнопку . В открывшемся диалоговом окне нажмите на кнопку , для подтверждения операции удаления.
Справочники. Управляющие программы. Удаление записи
Для того чтобы обеспечить поддержку версионности и хранения истории управляющих программ, нажмите на кнопку на панели навигации.
В открывшейся экранной форме, через панель навигации могут быть вызваны следующие операции:
- Добавить;
- Клонировать;
- Удалить;
- Обновить.
Нажмите на кнопку , чтобы добавить версию управляющей программы и указать необходимые параметры.
Справочники. Управляющие программы. Версии. Шаг 1
Для того чтобы изменить формат загружаемого файла, выберите необходимый параметр из выпадающего списка. Нажмите на кнопку или перетащите требуемый файл в область "Файл УП".
Справочники. Управляющие программы. Версии. Шаг 2
 
Если необходимо посмотреть текст управляющей программы или отредактировать ее, нажмите на кнопку . В открывшемся модальном окне введите все требуемые правки и нажмите на кнопку , для подтверждения сохранения внесенных изменений в тело управляющей программы.
Справочники. Управляющие программы. Версии. Шаг 3
 
В случае, если необходимо изменить статус управляющей программы и сделать ее актуальной - выберите из выпадающего списка пункт "Активна".
 
Примечание: При установке статуса "Активная версия" - нельзя будет отредактировать или удалить версию управляющей программы.
Справочники. Управляющие программы. Версии. Шаг 3
Для того чтобы клонировать управляющую программу, необходимо выбрать версию и нажать на кнопку на панели навигации. Появится уведомительное окно об успешном добавлении клонированной версии управляющей программы.
Справочники. Управляющие программы. Версии. Клонирование версии
 
Для удаления версии управляющей программы - нажмите на кнопку на панели навигации. В открывшемся диалоговом окне, нажмите на кнопку , чтобы подтвердить операцию.
Справочники. Управляющие программы. Версии. Удаление версии
 
Для того чтобы проследить историю внесенных ранее изменений по управляющей программе, нажмите на вкладку "История". На экранной форме отобразится вся необходимая сохраненная информация по внесенным правкам.
