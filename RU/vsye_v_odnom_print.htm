Всё в одном

Для проектов с количеством станков до пяти и пилотных проектов допустимо устанавливать компоненты на один сервер.
Рекомендуемые требования:
  • Windows Server 2016 Standard
  • MS SQL Server 2017 Standard x64
  • Процессор: частота 3ГГц, 16 ядер / 32 потока
  • Оперативная память: 32ГБ
  • Диски под ОС и компоненты DPA:     от 120ГБ SAS 10K.
  • Диски под базу данных DPA:     SSD от 200ГБ.
  • Сетевое подключение: 1Гб/с
 
Примечание.
Рекомендуем объединить диски под ОС в общий массив RAID1 (2 диска). Емкость RAID1 = половина от общей емкости дисков. Такой массив повысит отказоустойчивость дисковой системы.
Рекомендуем объединить диски под Базу Данных в общий массив RAID10 (четное количество, от 4х дисков). Емкость RAID10 = половина от общей емкости дисков. Такой массив обеспечит максимальную производительность и отказоустойчивость дисковой системы.
