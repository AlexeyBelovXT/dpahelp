Проекты до 70 станков

Для проектов, предназначенных для мониторинга ограниченного количества единиц оборудования, необходимо отдельно устанавливать DPA-сервер и компоненты системы: WEB-сервер (DPA Host), сервер БД (БД DPA) и Real-Time БД (индикаторы DPA);
При установке компонентов системы на отдельные сервера (подключение до 70 единиц оборудования), необходим следующий набор компонентов:
  • Операционная система: Windows Server 2016 Standard x64;
  • СУБД: Microsoft SQL Server 2017 Standard;
  • Процессор: частота 3.7/4.1 ГГц, 4 ядра/8 потоков, кэш 8 Мб;
  • Оперативная память: 16 Гб;
  • Жесткий диск: SATA 7200RPM, RAID 0, 480 Гб и выше – 2 шт.;
  • SSD-накопитель: SATA, 480 Гб, максимальная скорость записи 450 Мб/с, максимальная скорость чтения 380 Мб/с.;
  • Ethernet  - 1Гбит.
