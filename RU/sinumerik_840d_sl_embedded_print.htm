Первым шагом выбираем режим работы с системой, указываем пароль и получаем доступ к системным настройкам.
 
Для этого на экране "Ввод в эксплуатацию" (Start-up) жмем на кнопку "Set password" и указываем пароль для соответствующего уровня доступа.
Предустановленный пароль SUNRISE дает доступ на уровне системы.
 
Protection level    Protected by                                       Area
0                          Password                                           Siemens
1                          Password: SUNRISE (default)           Vendor
2                          Password: EVENING (default)           Service
3                          Password: CUSTOMER (default)      User
 
 
Для просмотра версий установленного оборудования выберите раздел "Диагностика" (Diagnosis)
и с помощью кнопок "Service Displays" → "Version" можно перейти на экраны просмотра версий NCU, HMI
 
 
Для изменения сетевых настроек на экране "Ввод в эксплуатацию" (Start-up) выберите опцию "Netw.Config."
 
Здесь можно указать настройки X130 интерфейса:
  • адрес статический либо по DHCP
  • Имя хоста (станка) в локальной сети
  • Настройки DNS серверов
    Все эти настройки необходимо уточнить у сетевого администратора
