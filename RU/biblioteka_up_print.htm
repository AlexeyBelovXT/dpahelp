Библиотека УП

 
Справочник «Управляющие программы» - централизованное версинное хранилище управляющих программ и связанных с ними нормативов.
Для того чтобы наполнить данный справочник - на панели навигации нажмите на кнопку , и в выпадающем списке выберите пункт «Справочники».
Главное меню. Стартовая страница
 
В открывшейся экранной форме существует несколько вариантов перехода к выбранному справочнику:
  • Выберите справочник «Типы причин простоя» и нажмите на кнопку   в левом углу экранной формы;
  • Выполните клик правой кнопкой мыши и выберите пункт «Перейти»;
  • Выполните двойной клик левой кнопкой мыши.
Справочники
 
 Аналогичным образом, через контекстное меню или панель навигации могут быть вызваны следующие операции:
- Добавить;
- Редактировать;
- Удалить;
- Перейти;
- Обновить.
На панели навигации нажмите на кнопку , для того чтобы добавить управляющую программу.

Примечание: Аналогичным образом можно отредактировать любую сохраненную управляющую программу, с помощью кнопки на панели навигации.
 
Для того чтобы удалить сохраненную управляющую программу из перечня, выберите требуемый пункт и нажмите на кнопку . В открывшемся диалоговом окне нажмите на кнопку , для подтверждения операции удаления.
