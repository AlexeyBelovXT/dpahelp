Security

 
For the enterprise level, the production site, department and equipment access settings can be set. By default all users have an access to all structure items.
For setup select the corresponding structure item, on Security tab, select a checkbox "Only to the specified groups" and click "Edit".
 
Note:You cannot change right of access so far the option "Inherit Settings" selected
                                                                                                                              Selected structure item. Security tab
Select appropriate group (s) and click "Select".

Click "Save" .
