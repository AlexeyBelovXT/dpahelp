Базовые настройки

 
Первым шагом подключаем к DPA Host предварительно установленный DPA Server, затем добавляем лицензию и оборудование.
 
После успешной установки системы на рабочем столе создаются ярлыки «CONFIG» и «HOST».
Перейдите по ссылке «HOST». На панели навигации нажмите на кнопку , и из выпадающего списка выберите пункт «Мониторинг».
Главное меню. Стартовая страница
 
Если установка компонентов произведена на один сервер, то DPA Server и транспорт будут добавлены автоматически.
Если DPA server устанавливали отдельно, переходите к следующему разделу.
