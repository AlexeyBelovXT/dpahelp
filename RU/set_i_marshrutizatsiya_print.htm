Сеть и маршрутизация

Требования к сети:
Сервер или ПК, на котором установлен компонент DPA, должен подключаться к сети через сетевую карту со скоростью 1Гб/с.
 
Если в сети установлено несколько DPA Server, то на машину с DPA Host рекомендуем установить сетевую карту со скоростью 10Гб/с.
 
Если DPA Server и DPA Host установлены на разные машины, то:
  • со стороны DPA Host необходимо открыть порт TCP 45617;
  • со стороны DPA Server необходимо открыть порты TCP 45618 и UDP 1000-10000.    
 
Примечание:
  • Во время установки компонент DPA автоматически создаются правила для стандартных файерволов Брандмауэр Windows / Windows Firewall или Защитник Windows / Windows Defender.
  • Если используете другое ПО для защиты сетевых подключений, обратитесь к системному администратору.
  • Попросите сетевого администратора проверить, что машины с компонентами DPA могут устанавливать сетевое подключение к оборудованию (станкам с ЧПУ).
