Подготовка сервера к установке

 
Для развертывания компонентов системы требуется производительный сервер с установленной 64-x разрядной операционной системой (ОС) семейства Microsoft Windows.
 
Проверьте сетевые настройки. Компоненты DPA могут быть развернуты на нескольких машинах. Убедитесь, что все сервера подключены к одной локальной сети, имеют необходимые настройки.
 
Подготовьте сервер баз данных1. Подробную инструкцию по установке MS SQL можно посмотреть на сайте Микрософт.
Обратите внимание, что консоль управления SQL сервером не идет в комплекте. Установщик необходимо загрузить с официального сайта разработчика.
Консоль можно развернуть как на сервере SQL, так и на другой машине (например ПК администратора). Убедитесь, что можно подключиться через консоль управления к серверу SQL.
 
Требования к дополнительному программному обеспечению:
  •  NET Framework 4.6.2 (https://www.microsoft.com/ru-ru/download/details.aspx?id=53344);
     
В ходе установки DPA будет создана и настроена база данных, а также запущены службы приложений. Описанные действия могут быть выполнены от имени различных типов учетных записей:
· Localsystem. Встроенная в операционную систему учетная запись. По умолчанию обладает самыми высокими привилегиями на сервере;
·  Интегрированная учетная запись (Windows). Установка системы, создание и настройка базы, а также запуск служб может производится от имени учетной записи, созданной на сервере или в домене Active Directory;
·  Пользователь SQL. Учетная запись создается с помощью консоли управления SQL сервером. У такой записи есть доступ только к компонентам и службам соответствующего SQL сервера.   
 
Не рекомендуем использовать встроенные хорошо известные учетные записи. Помимо известных имен, встроенные учетные записи обладают избыточными (сверх необходимого) правами и доступами, которые не требуются для работы DPA. Безопаснее создать отдельные учетные записи и использовать их для установки компонент DPA.
 
Убедитесь, что соответствующие пользователи обладают необходимыми и достаточными (не сверх того) полномочиями.
