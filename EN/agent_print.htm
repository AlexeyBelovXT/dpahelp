Agent

 
Connection of some drivers requires installation of Agent, this process is described in the instruction for setup CNC specific type.
If necessary updating of the agent can be executed from the server, without stopping the machine. 
Note: updating of the agent can be made only on the switched-on machine!
Click , select Monitoring
Select the required driver and check a status.
Мониторинг. Включенный драйвер
 
Click «Agent» tab.
In line "Version" is reflected the version of the agent installed on the machine.
On a control bar in the Available Installers on the Machine area press , where the installation files of agents which are earlier loaded on the machine are displayed.
Monitoring settings. Install files for agent
In the opened window select the necessary version of the agent and press "Select" .
Monitoring. Agent. Install Files
The system will inform you on a successful completion of operation, the agent to appear in the list. Further select the necessary version of the agent and click.
After emergence of the message "Installation is complete" in window "Installation of the Agent" click "Close".
The agent version will be updated after the agent service is restarted. Process of restart can take up to 10 minutes. In the course of restart the machine will be unavailable to a monitoring system.
 
Note: Loading of installation filesis  made at regular installation of a system, further for updating of the agent download the installation file on ftp://ftp.x-tensive.com/Public/DPA%20Agents/ and copy to  «%\ProgramData\X-tensive\DPA\Installers» on DPA server.
