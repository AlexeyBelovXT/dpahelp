Sinumerik 828D Operate

 
Для просмотра информации о системе и редактирования настроек надо перейти в режим администрирования
 
1) Start-up
2) Password
3) Set password
4) указываем один из паролей по умолчанию.
 SUNRISE (предустановленный) уровень производителя
 EVENING (предустановленный) уровень сервисного обслуживания
 CUSTOMER (предустановленный) уровень пользователя
 
 
Для изменения сетевых настроек:
1) Start-up
2)
3) Network
4) Overview
5) Change
Для изменения сетевых настроек подключения к Локальной сети компании выбираем
6) Company
7) Change
 
В настройках можно указать:
Шлюз по умолчанию
DNS сервера
Сервер времени
Имя хоста
Настройки файрвола:
порт для взаимодействия через STEP7
порт для доступа по VNC
любые дополнительные порты
 
Проверить свои настройки можно с помощью меню Диагностики
1) Diagnostic
2) Bus tcp/ip
3) tcp/ip diagnostic
4) Details
будут отображены текущие сетевые настройки.
