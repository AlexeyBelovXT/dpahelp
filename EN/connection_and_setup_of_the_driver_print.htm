Connection and setup of the driver

 
The Driver is the type of equipment defined and supported by a system and the version of the CNC. For each driver in DPA the protocol of data acquisition and setup, containing the description of the data requested by a system from a rack is defined.
In that specific case the Driver is a CNC machine with unique IP.
 
In certain cases it is necessary to install the Agent on the machine . Download the installation file according to instructions in the section "Driver Creation".
