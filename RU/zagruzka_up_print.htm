Загрузка УП

 
При работе на станках с ЧПУ существует возможность обратиться к сетевому хранилищу УП для загрузки корректной программы на станок.
 
Примечание: Для работы в модулях "Загрузчик УП" и "Оператор" у пользователя системы должны быть соответствующие права доступа к функциям.
 
  • Загрузка УП оператором – для загрузки УП на станок оператором;
  • Управление задачами для создания заказов
• Операторы для доступа к модулю Оператор
  • Загрузка УП оператором – для загрузки УП на станок оператором
 
 
Для работы во всех перечисленных модулях у пользователя должны быть соответствующие права доступа к функциям.
 
 
Безопасность. Права доступа
 
Система проверяет имя и тело, активной, в данный момент УП (при ее наличии), и в случае расхождения, рекомендует мастеру/технологу загрузить файл программы из хранилища на сервере.
Существует два способа загрузки управляющей программы в память станка:
  • Кнопка «Загрузить» служит для загрузки УП на карту памяти станка;
  • Кнопка «Загрузить и выбрать» служит для загрузки активации УП (активация означает пометку файла УП, как активного, для последующего быстрого запуска оператором).
