Установка DPA Server

Чтобы скачать инсталлер для установки DPA Server: на стартовой странице нажмите на кнопку и из выпадающего списка выберите пункт "Установочные файлы".
Главное меню. Стартовая страница. 
На экранной форме "Установочные файлы" выберите пункт "DPA Server" из представленного перечня, и нажмите на кнопку . Файл Setup.DPA.Server.(версия).exe будет сохранен в указанную директорию.
Главное меню. Стартовая страница. 
Запустите приложение Setup.DPA.Server.(версия).exe.
                                        Мастер установки. Начало
 
Нажмите на кнопку .
Внимательно прочитайте условия лицензионного соглашения. Если Вы согласны с условиями, поставьте флаг и нажмите кнопку , в противном случае – кнопку .
                        Мастер установки. Лицензионное соглашение
 
Проверьте необходимые требования к свободному месту на диске, если необходимо измените директорию установки.
Примечание: Не меняйте параметры, установленные по умолчанию, без необходимости.
 
Нажмите кнопку .
       
                                  Мастер установки. Проверка требований
 
                             Мастер установки. Смена директории установки
 
Система запрашивает от имени какого пользователя должен производиться запуск службы DPA Server.
Примечание: Необходимо убедиться, что выбранный пользователь обладает необходимыми полномочиями.
 
Нажмите кнопку .
                             Мастер установки. Конфигурация DPA Server
 
Система запрашивает подтверждение параметров установки. Нажмите кнопку .
                           Мастер установки. Подтверждение установки
 
Ожидайте, идет установка.
                                           Мастер установки. Установка
 
Поздравляем, система успешно установлена.
                            Мастер установки. Завершение установки
 
Нажмите на кнопку , для подтверждения факта завершения установки DPA Server.
