Создание драйвера

 
Существует два способа формирования драйвера:
  • Правой кнопкой мыши вызвав контекстное меню, выберите «Добавить/Драйвер»:
                                                                                                                                                    Инициация создания драйвера через контекстное меню
Нажмите на кнопку на панели навигации в левом верхнем углу экранной формы:
                                                                                                                                              Инициация создания драйвера через панель навигации
 
Возможные варианты подключения:
                                                                                                                                                                      Варианты создания драйвера
 
1. При выборе пункта «Вручную», необходимо будет вручную указать параметры подключения;
2. «Импорт» применим в случаях, когда требуемый драйвер не предустановлен в системе;
3. «Serial» позволяет создать драйвер для специфичных ЧПУ, подключаемых через COM-порты;
4. «Ethernet», необходимо воспользоваться либо автоматической настройкой формирования драйвера, либо указывать необходимые параметры установки для создания драйвера вручную.
 
Далее описаны варианты для самостоятельного формирования драйвера:
·     Создание драйвера с предзаполнением всех полей в модальном окне «Создание драйвера», позволит ускорить процесс формирования драйвера. Нажмите на кнопку  . 
                                                                                                                                         Создание драйвера через Ethernet. Шаг 11
 
Ожидайте. Система запрашивает параметры единицы оборудования по указанному IP-адресу.
                                                                                                                                              Создание драйвера через Ethernet. Шаг 2
Система обнаружила драйвер на указанном IP-адресе. Нажмите на кнопку .
                                                                                                                                                      Создание драйвера через Ethernet. Шаг 3
 
Для создания драйвера выберите пункт «Из существующих», и в выпадающем списке, выберите необходимую конфигурацию драйвера.
                                                                                                                                                Создание драйвера через Ethernet. Шаг 4
Выберите требуемый транспорт, установите флаг и нажмите на кнопку .
                                                                                                                                              Создание драйвера через Ethernet. Шаг 5
 
Введите наименование оборудования. По умолчанию в системе описан простейший план предприятия, оборудование должно быть закреплено за конкретным подразделением. В последующем, после конфигурации структуры предприятия, выбор может быть изменен. 
                                                                                                                                                    Создание драйвера через Ethernet. Шаг 6
Подтвердите создание драйвера, нажав на кнопку .
                                                                                                                                                  Создание драйвера через Ethernet. Шаг 72
·     Создание драйвера в автоматическом режиме. В модальном окне "Создание драйвера" заполните поле "IP адрес" и нажмите на кнопку  .
                                                                                                                                                      Создание драйвера через Ethernet. Шаг 1
 
Ожидайте. Система запрашивает параметры единицы оборудования по указанному IP-адресу.
                                                                                                                                                   Создание драйвера через Ethernet. Шаг 2
Остальные пункты формирования драйвера в автоматическом режиме аналогичны пунктам, приведенным выше, для формирования драйвера в предзаполненном режиме.
 
К одному DPA-серверу может быть подключено несколько станков. Определение оптимального количества подключаемого оборудования описано в инструкции по установке системы.
В случае подключения оборудования, подключаемого через COM-порт, перечень указываемых параметров специфичен и включает параметры COM-портов ЧПУ и сервера.
 
Пример заполнения и источник данных (BaudRate – бит в секунду, Parity - четность, DataBits - биты данных, StopBits – стоповые биты, Hadshakes – управление потоком):
                                                                                                                    Пример параметров для настройки драйвера, подключенного через COM-порт
