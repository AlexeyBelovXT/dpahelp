Установка DPA

Загрузите актуальную версию установщика с нашего сайта. Запустите приложение Setup.DPA.(версия).exe.
1. Выберите язык из выпадающего списка, который будет использоваться в процессе установки.
                                                                       1. Мастер установки. Начало
Нажмите на кнопку "Далее".
2. Внимательно прочитайте условия лицензионного соглашения.
Если Вы согласны с условиями, нажмите кнопку "Принимаю". Для завершения установки нажмите кнопку "Отмена".
                                                       2. Мастер установки. Лицензионное соглашение
 
3. По умолчанию предлагается установить все компоненты DPA в стандартные папки.
Если необходимо установить только одну из компонент, снимите лишнюю галочку.
Выбрать другую папку для установки можно с помощью кнопки .
Примечание: Не меняйте параметры, установленные по умолчанию, без необходимости.
 
Нажмите кнопку "Далее".
                                                            3. Мастер установки. Выбор компонентов 
 
4. Система запрашивает от имени какого пользователя должен производиться запуск службы DPA Host.
 4.1 Мастер установки. Конфигурация DPA Host
 
Так же запрашиваются данные для подключения к базе данных: имя экземпляра, имя существующей или вновь создаваемой базы данных.
С помощью оснастки Службы на сервере, где установлен MS SQL (или другая СУБД) можно узнать имя.
 4.2 Имя экземпляра базы данных указано в скобках рядом с именем службы SQL Server
 
По умолчанию система предлагает создать базу данных DPA от имени пользователя, который запустил установку системы. Вы можете указать другой способ авторизации/уполномоченного пользователя, а также выбрать существующую базу данных или создать новую, с помощью кнопки .
 4.3 Мастер установки. Конфигурация DPA Host. Выбор базы данных
Нажмите на кнопку "Создать", для того чтобы зафиксировать выбранные параметры. Нажмите на кнопку "Далее".
 
5. На этом шаге система подтверждает выбранные настройки. Вы можете внести изменения и применить настройки, нажав кнопку "Настройка базы данных".
                                                       5. Мастер установки. Настройка базы данных
Если сразу нажмете Пропустить, программа установки применит текущие настройки на финальном шаге Установка, после прохождения всех шагов по выбору настроек.
 
6. Система содержит простейшую предустановленную структуру.
Проставьте флаги напротив тех элементов, которые необходимо будет добавить в структуру объекта автоматизации:
  •  Добавить виртуальный цех (будет создано предприятие, производственная площадка (цех), подразделение (участок/линия) и оборудование (рабочий центр), также будет заполнен справочник "3D-модели оборудования" и добавлена стандартная мнемосхема);
  •  Добавить изделие, управляющую программу и технологию (будут заполнены справочники "Изделия", "Тип изделия", "Управляющие программы" и "Тип УП", а также добавлена технология изделия;
  •  Добавить пользователей и группы (будет добавлен минимально необходимый редактируемый перечень групп и пользователей системы);
Примечание: Пропустите этот шаг, если вы устанавливаете DPA в производство, не для тестирования. Достаточно не ставить галочки и сразу нажать "Далее".
6. Мастер установки. Предустановленные данные
 
7. Система запрашивает от имени какого пользователя должен производиться запуск службы DPA Server.
Примечание: Необходимо убедиться, что выбранный пользователь обладает необходимыми полномочиями.
Выберите необходимые настройки и нажмите кнопку "Далее". .
                                                        7. Мастер установки. Конфигурация DPA Server
8. Система запрашивает подтверждение параметров установки.
Примечание. Если сомневаетесь в правильности выбора настроек, проконсультируйтесь у специалистов X-tensive.
Если все данные верны, нажмите кнопку .
                                                     8. Мастер установки. Подтверждение установки
 
9. Ожидайте, идет установка.
 
                                                                      9. Мастер установки. Установка
 
10. Поздравляем, система успешно установлена.
                                                            10. Мастер установки. Завершение установки
