Загрузка установочных компонентов системы

На панели навигации нажмите на кнопку , и из выпадающего списка выберите пункт «Установочные файлы».
                                                                                                                                           Главное меню. Установочные файлы
 
На экранной форме «Установочные файлы» выберите файл, который необходимо скачать для дальнейшей установки программного обеспечения.
Инициация загрузки установочных файлов
Для того инициировать скачивание файла, кликните дважды кнопкой мыши на необходимый пункт, либо нажмите на кнопку в левой части экранной формы.
Скачивание установочных файлов
 
Система сгенерирует файл c разрешением «.msi» и сохранит его в папке «Загрузки» браузера.
