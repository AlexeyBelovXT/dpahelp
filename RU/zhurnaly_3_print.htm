Журналы

 
Модуль «Журналы» предназначен для архивных данных мониторинга в табличном виде. 
На панели навигации нажмите на кнопку , и в выпадающем списке выберите пункт «Журналы».
Главное меню. Стартовая страница
 
Запись данных в журнал происходит по факту изменения одного из протоколируемых параметров.
Для перехода к выбранному журналу:
•   выполните двойной клик левой кнопкой мыши на заголовок;
•   выполните клик правой кнопкой мыши на заголовке и нажмите на кнопку ;
•   выполните клик левой кнопкой мыши на заголовке и выберите пункт "Перейти" из выпадающего списка.
